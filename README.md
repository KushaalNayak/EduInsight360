# EduInsight360 – Student Performance Analytics Portal

EduInsight360 is a premium web application designed for faculty and students to track academic performance, generate reports, and analyze growth trends.

## 🏗️ Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Charts**: Recharts
- **Styling**: Glassmorphism & Modern UI/UX Design

## 🚀 Quick Run (Recommended)

The project is fully containerized. To run both the frontend and backend with a single command:

```bash
docker-compose up --build
```

- **Frontend:** [http://localhost:5173](http://localhost:5173)
- **Backend:** [http://localhost:8081](http://localhost:8081)
- **Default Auth:** `kushaal` / `password123`


## 🛠️ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/KushaalNayak/EduInsight360.git
   cd EduInsight360
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 👨‍🏫 Key Features

- **Staff Portal**: Faculty can manage student lists, monitor class averages, and view "at-risk" student alerts.
- **Student Portal**: Students can view their marks, subject trends, and personalized academic recommendations.
- **Interactive Analytics**: Rich data visualizations using Recharts to track semester-wise performance.
- **Secure Access**: Role-based portal experience for both Staff and Students.
- **Responsive Design**: Fully optimized for both desktop and mobile viewing.
